HTML 基本介紹

HTML（HyperText Markup Language）是用於建立網頁的標準語言之一。它由一系列標記（或稱為標籤）組成，每個標記描述了網頁上的一個元素或內容。

HTML 文件結構

HTML 文件的基本結構通常包含以下部分：

文檔類型聲明（Document Type Declaration）

文檔類型聲明（DOCTYPE）告訴瀏覽器正在使用哪個 HTML 版本來解析文件。

HTML 元素

  • <html>：整個 HTML 文件的根元素，包含了所有其他的 HTML 元素。
  • <head>：用於定義網頁的元信息，例如標題、連結和腳本等。
  • <title>：設置網頁的標題，顯示在瀏覽器的標題欄中。
  • <body>：包含了可見的網頁內容，例如文本、圖片、連結等。

HTML 標籤

HTML 標籤用於定義網頁上的不同元素。它們通常是成對出現的，包括開始標籤和結束標籤。

  • <p>：段落標籤，用於定義一個段落。
  • <a>：錨點標籤，用於創建連結。
  • <img>：圖片標籤，用於插入圖片。

屬性

HTML 標籤可以包含屬性，這些屬性提供了有關元素的更多信息。

  • href：用於指定連結的 URL。
  • src：用於指定圖片的 URL。
  • alt：用於定義圖片的替代文本。
  • width 和 height：用於設置圖片的寬度和高度。

總結

這僅僅是 HTML 的入門介紹，HTML 還有許多其他元素和屬性，用於創建豐富的網頁。

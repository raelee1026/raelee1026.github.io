CSS 基本介紹

CSS（Cascading Style Sheets）是一種用於設置網頁樣式和外觀的語言。它可以將 HTML 元素的外觀（如顏色、大小、間距等）進行美化和定制。

CSS 樣式規則

CSS 通過樣式規則（style rules）來定義網頁元素的外觀。每個樣式規則由兩部分組成：選擇器（selector）和聲明塊（declaration block）。

  • 選擇器（selector）：用於選擇要應用樣式的 HTML 元素。可以是元素名稱、類名、ID 等。
  • 聲明塊（declaration block）：包含一個或多個屬性-值對，用於定義元素的樣式。

CSS 屬性和值

CSS 屬性用於指定要修改的屬性，而值則用於定義屬性的具體設置。

範例

  • color：指定文本顏色，可以使用顏色名稱、十六進制代碼、RGB 值等。
  • font-size：指定字體大小，可以使用像素（px）、百分比（%）等。
  • background-color：指定背景顏色，同樣可以使用顏色名稱、十六進制代碼等。
  • padding：指定元素內容與邊框之間的距離。
  • margin：指定元素與其他元素之間的距離。

HTML、CSS 和 JavaScript 的關係

HTML 用於定義網頁的結構和內容，CSS 用於設置網頁的外觀和樣式，而 JavaScript 用於實現網頁的動態功能和交互。這三種語言共同工作，使得我們能夠創建豐富、互動性強的網頁。

總結

這只是 CSS 的入門介紹，CSS 還有許多其他屬性和功能，可以用於創建豐富的網頁樣式。

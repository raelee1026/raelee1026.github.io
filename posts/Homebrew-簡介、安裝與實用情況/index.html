Homebrew 簡介、安裝與實用情況

什麼是 Homebrew？

Homebrew 是 macOS 和 Linux 上的一個包管理工具，它讓用戶可以方便地安裝、管理和更新各種軟體包。透過 Homebrew，你可以輕鬆地從 terminal 安裝你所需的軟體，讓你不必在網路上下載安裝包自行安裝。

誰適合安裝 Homebrew？

如果你經常需要安裝各種開發工具，那麼 Homebrew 將會是你不可或缺的工具。
（如果平常沒有使用需求，也可以先安裝好！！）

Homebrew 的實用情況舉例

  • 安裝開發工具：想要安裝 Node.js 或 Python？只需一行命令：brew install node 或 brew install python。
  • 管理多版本軟體：Homebrew 也支援安裝多個版本的軟體，如 brew install node@14 安裝指定版本的 Node.js。

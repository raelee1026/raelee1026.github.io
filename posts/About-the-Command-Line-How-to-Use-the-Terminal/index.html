About the Command Line - How to Use the Terminal

Command Line 是作業系統中一個強大且神秘的工具，它為使用者提供了直接與電腦交互的途徑。

為什麼需要 Terminal？

  • 效率： Terminal 有時候更有效率，不用在不同視窗間切換點擊，幫助我們節省時間，並能夠更快地執行任務。

  • 自動化任務： 可以自動化任務，使用 shell 構建腳本，然後隨時執行這些腳本。當處理我們不想一遍又一遍地重複的重複性任務時，這是非常有用的。

  • 特殊情況： 有時只能使用 Terminal，沒有其他方法。

關於 Shell

Shell 是一個充當命令行解釋器的程式。它處理指令並輸出結果。它解釋並處理使用者輸入的指令。 簡單來說，Shell 是一個命令行解釋器，而 Terminal 是用戶與 Shell 進行交互的界面。通常，我們會在 Terminal 運行 Shell，通過 Shell 執行命令和管理系統。

要知道目前電腦正在運行哪個 Shell，只需打開終端並輸入echo $0。

Bash 是最廣泛使用的，並且預設安裝在 Mac 和 Linux 上。 Zsh 與 Bash 非常相似，但它是在 Bash 之後創建的，並且在某些方面有一些改進。還有一些其他的 Shell，如 Fish、Ash、Dash 等。

常見指令

1. ls / dir

用於列出當前目錄下的文件和目錄。

2. cd

用於改變當前工作目錄。

3. pwd

用於顯示當前工作目錄的路徑。

4. touch

用於創建新的空文件。

5. rm / del

用於刪除文件。

6. mkdir

用於創建新的目錄。

7. rmdir

用於刪除目錄。

8. cat / type

用於顯示文件的內容（但通常應該會直接打開吧）。

9.mv / move

10. cp

用於複製文件或目錄。

11. echo

顯示文本或變數值。

12. history

用於顯示最近執行的指令歷史記錄。

13. How to compile?

用於將源代碼編譯成可執行文件或目標代碼。

顯示類似《駭客帝國》中的綠色矩陣效果。
